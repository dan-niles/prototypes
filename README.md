# Mockups

A personal mockup viewer hosted on GitHub Pages. Each mockup lives at its own URL and supports multiple versions for tracking design iterations.

Built with React, TypeScript, Tailwind CSS v4, and Vite. Designed to work with AI-generated mockups from tools like Figma AI and v0 — drop the output into a folder, register it, and it's live.

**Live site:** [dan-niles.github.io/mockups](https://dan-niles.github.io/mockups)

For a full breakdown of how everything works, see [PROJECT.md](./PROJECT.md).
